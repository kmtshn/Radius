# Radius - AI Security Monitor

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)
![Technology](https://img.shields.io/badge/TensorFlow.js-Powered-orange.svg)

**Radius** は、Webブラウザだけで動作する、プライバシー重視のAI監視カメラシステムです。
特別な機材やサーバーは不要。PCやタブレットのWebカメラを使い、指定したエリアへの人物侵入をリアルタイムで検知・記録します。

![App Screenshot](./screenshot.png)
## ✨ 特徴 (Features)

* **⚡ リアルタイムAI検知:** TensorFlow.js (Coco-SSD) を使用し、ブラウザ上で高速に人物を検知します。
* **🔒 完全プライバシー保護:** 映像データはすべて端末内で処理されます（Edge Computing）。**外部サーバーへの送信は一切ありません。**
* **🎨 直感的なエリア設定:** 監視したい「立ち入り禁止エリア」を画面上でドラッグ＆ドロップして自由に設定可能。
* **📸 自動証拠撮影:** 侵入を検知した瞬間、自動で写真を撮影しログに保存します。
* **📂 ギャラリー & エクスポート:**
    * 撮影された写真はアプリ内のギャラリーで確認可能。
    * 気に入った写真を個別にダウンロード、または**ZIP形式で一括ダウンロード**できます。
* **💾 ハイブリッド保存システム:**
    * **ローカルサーバー実行時:** IndexedDBを使用し、ブラウザを閉じてもデータが永続化されます。
    * **直接ファイル実行時:** メモリ一時保存モードで動作し、インストール不要ですぐに使えます。
* **💎 Modern Clean Canvas Design:** 洗練されたUIで、プロフェッショナルな監視体験を提供します。

## 🛠 使用技術 (Tech Stack)

* **Frontend:** HTML5, JavaScript (Vanilla ES6+)
* **Styling:** Tailwind CSS (CDN)
* **AI Model:** TensorFlow.js / Coco-SSD
* **Storage:** IndexedDB (Wrapperなしのネイティブ実装)
* **Utilities:** JSZip (画像の一括圧縮・出力)

## 🚀 使い方 (How to Run)

このアプリは単一のHTMLファイルで構成されています。

### 推奨: ローカルサーバーで実行する場合
データの永続保存（ブラウザを閉じても写真が消えない）を利用したい場合は、ローカルサーバー経由で開いてください。

1.  このリポジトリをクローンまたはダウンロードします。
2.  VS Codeの「Live Server」拡張機能などを使用し、`index.html` を開きます。
3.  ブラウザで `http://127.0.0.1:5500` 等にアクセスします。

### 簡易: ファイルを直接開く場合
今すぐ試したい場合は、ファイルをダブルクリックするだけで動作します。

1.  `index.html` をブラウザ（Chrome/Edge推奨）で直接開きます。
2.  **注意:** セキュリティ制約によりデータベースが使用できないため、「一時保存モード」で動作します（リロードすると写真は消えます）。

## 📖 操作マニュアル

1.  **起動:** 「カメラを起動」ボタンを押して監視を開始します。
2.  **エリア設定:**
    * 「エリア設定」ボタンを押すと編集モードになります。
    * 画面上の黄色い枠をドラッグまたはリサイズして、監視したい範囲を指定します。
    * 「設定完了」を押して監視を再開します。
3.  **監視中:**
    * 指定エリアに人が侵入すると、警告アラートが表示され、音声で警告します。
    * 同時に自動で写真が撮影されます（3秒のクールダウンあり）。
4.  **ログ確認:**
    * サイドバーの「ログ履歴 (ギャラリー)」をクリックすると、撮影された写真一覧が表示されます。
    * 写真をクリックで拡大、右上のアイコンで個別保存、「すべて保存(ZIP)」で一括保存が可能です。
